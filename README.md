# youtube-sentiment-analysis
# 📊 Sentiment Analysis of YouTube Comments  
### Case Study: Dhruv Rathee’s “Dhurandhar 2” Video

---

## 📌 Project Overview
This project analyzes audience sentiment on a YouTube video using Natural Language Processing (NLP).  
The goal is to understand how viewers react to opinion-based content and identify patterns in engagement behavior.

---

## 🎯 Objectives
- Analyze sentiment distribution (Positive, Negative, Neutral)
- Identify engagement patterns in user comments
- Explore relationship between sentiment and comment length
- Derive insights from audience behavior

---

## 🛠️ Tools & Technologies
- Python  
- Pandas  
- Matplotlib  
- analyzer (Sentiment Analysis)  
- Jupyter Notebook  

---

## 📂 Dataset
- Source: YouTube comments (extracted and stored in CSV format)
- Total comments analyzed: ~60,000+
- Data includes raw comment text

---

## 🔄 Workflow

1. Data Collection  
2. Data Cleaning  
3. Sentiment Classification  
4. Data Visualization  
5. Insight Generation  

---

## 📊 Key Findings

### 🔹 Sentiment Distribution
- Majority of comments are classified as **Neutral**
- Positive comments form the second largest group
- Negative comments represent a smaller but significant portion

### 🔹 Engagement Behavior
- Negative comments have the **highest average length**
- Positive comments are moderately long
- Neutral comments are shortest and reflect low engagement

### 🔹 Interpretation
- Dhruv Rathee's content generates **high emotional engagement**
- Strong opinions (especially negative) lead to longer, more detailed responses
- Neutral sentiment dominance may be influenced by model limitations

---

## 📈 Visualization
- Bar chart showing sentiment distribution
- Comparative analysis of comment lengths by sentiment

---

## ⚠️ Limitations
- Comments may not accurately capture sarcasm, slang, or mixed-language comments
- Neutral sentiment may be overrepresented due to model limitations

---

## 🚀 Future Improvements
- Use VADER or transformer-based models for better sentiment accuracy
- Filter short/low-information comments
- Perform keyword/topic analysis
- Build an interactive dashboard (Power BI)

---

## 🧠 Conclusion
The analysis reveals that while most comments appear neutral, deeper engagement is driven by emotional responses, especially negative sentiment.  
This highlights the polarized nature of audience interaction in political content.
